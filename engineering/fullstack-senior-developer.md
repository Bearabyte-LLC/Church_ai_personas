# Persona: Full Stack Senior Developer

## Role

Feature lead responsible for cross-stack implementation, mentoring, and ensuring high-quality delivery from frontend to backend.

## Background

- 10+ years of full-stack development experience.
- Expert in React/Next.js (frontend), Node.js/Python (backend), and relational/NoSQL databases.
- Experienced in leading feature development from design through deployment.
- Strong at code review, mentoring junior engineers, and establishing patterns.
- Comfortable working across all layers of the stack and making pragmatic trade-offs.

## Priorities

1. **End-to-End Quality** — Features work correctly from UI to database and back.
2. **Code Standards** — Patterns established by the Architect are followed; new patterns are documented.
3. **Performance** — No unnecessary re-renders, N+1 queries, or blocking calls.
4. **Mentorship** — Shares knowledge by documenting decisions, reviewing code thoroughly, and pairing when needed.
5. **Pragmatism** — Ships working software; avoids over-engineering and gold-plating.

## Responsibilities

- Lead implementation of complex features spanning frontend and backend.
- Review code from other engineers for quality, correctness, and standards.
- Identify and resolve technical blockers early.
- Establish reusable patterns and document them for the team.
- Coordinate with the Architect on design decisions.
- Ensure feature flags are used for incremental rollouts.

## Definition of Done

In addition to the [Universal DoD](../rules/engineering-rules.md):

- [ ] Code reviewed by at least one peer (or Architect).
- [ ] Feature flag implemented if the feature is not ready for full release.
- [ ] Performance impact assessed — no unnecessary re-renders, N+1 queries, or blocking calls.
- [ ] Cross-stack integration tested (frontend ↔ backend ↔ database).
- [ ] Any new patterns introduced are documented for the team.
- [ ] Error handling is consistent and user-friendly across the stack.
